# xuexixitong
这是一个关于个人学习系统的构建和迭代项目，不包含任何代码。本项目坚持真实原则，宁肯不写也不能写虚假的内容上去。如果对你能有所帮助那是再好不过，欢迎你和我一起交流如何更好的学习。
## 一、我的学习环境与休息管理
### 1.电子产品的使用

1.1我把我手机上所有的可能让我上瘾的软件卸载了（我曾经有过七八年的成瘾史，包括但不限于刷视频，看网文，玩游戏，看漫画……事实上，经过我好几年的尝试戒瘾发现，把所有可能让你成瘾的东西变得难以接触是很好的）如哔哩哔哩、百度贴吧、小红书、抖音、快手、小黑盒等，如果我需要使用就需要到电脑上用。对于一些我学习需要用但可能让我成瘾的软件，我会用一些软件限制使用（我个人用的是appblock，限制只有在某个时间段才能使用）

1.2手机我会把需要的常用的学习软件放桌面上，不常用的或者与学习无关的不放在桌面

1.3手机和电脑我都进行了相应的改造，使用灰色等低饱和颜色，避免过于鲜艳的颜色对自己学习的影响

1.4每次打开电脑我都会先明确自己要做什么，使用的时候反复暗示自己要做什么防止自己去做别的事情

1.5我手机蓝光过滤全天开启拉到最大，每天到了晚上设置手机开启黑白模式和深色模式（一般是晚上八点半）


### 2.关于精力管理
2.1我每天保证睡满八个小时，作息如无特殊情况保持固定（晚上十点到第二天六点）

2.2我每天吃完午饭后半个小时都会午睡，午睡时间无特殊情况固定半个小时

2.3每天早上起来洗冷水澡，晚上洗热水澡

2.4每天晚上睡前都会给第二天列要做的事情清单，把自己要做的事情安排好后写上去。遇到什么要做的事情也及时记在清单上面防止忘记。做完一件事就打勾


### 3.学习环境布置
3.1我在学习时会找一个安静无人打扰的环境。我一般都是只在家里学习，在家里我有自己的房间，当我学习的时候关上大门（我和父母说过当我关上大门的时候如果没有重要的事情不要进来，更不能时不时打开门偷窥我学习）

3.2我的桌面只放和学习有关的东西，整齐摆放，学习的时候视野中心只放当前做的任务相关书籍

3.3在我的房间里我有一个专门的椅子，坐在这个椅子上我只能学习，且一旦坐上去就必须开始学习

3.4天气热的时候开空调。天气冷的时候多穿点不开空调

3.5学习的时候电子产品放到学习时看不到碰不到的地方

## 二、学习（目前主要是高中学习）
### 1.输入
1.1上课听讲，关键的笔记记下来

1.2把书上的定义用荧光笔划出来


### 2.内化
2.1背书的时候拆成最小单位，一个个去背（比如背古诗就是一个字一个字背，然后是一句、两句）

### 3.输出
3.1刷题，做错的仔细分析错因，错了好几次的同类型题目记在错题本上

### 4.学习习惯
4.1每天坚持背单词
4.2学习的时候使用番茄钟50—10的学习
